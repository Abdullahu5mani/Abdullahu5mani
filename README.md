## Projects

### [Taurscribe](https://github.com/Abdullahu5mani/Taurscribe)
Local transcription desktop app — no cloud, no data sent anywhere. Built with Rust, Tauri, TypeScript, and whisper.cpp. Runs Whisper inference on-device. Dealt with cross-platform MSVC linking and Windows CI/CD build issues.

`Rust` `TypeScript` `Tauri` `Whisper`

---

### [Neuro Co-Scientist](https://github.com/Abdullahu5mani/Bench-spark-Hackathon-Buraydah-) — BenchSpark Hackathon 2026
Agentic RAG system for neuroscience research. Processed ~30,000 PubMed papers, integrated ChEMBL drug data and AlphaFold protein structures into a unified vector search pipeline. Built on Google Vertex AI, BigQuery, and text-embedding-004.

`Python` `Google Cloud` `Vertex AI` `BigQuery` `RAG` `Vector Search`

---

### [Music Electron App](https://github.com/Abdullahu5mani/Music-Electron-App)
Desktop music app with a concurrent Node.js worker pool for CPU-intensive audio analysis and local AI model inference — keeps the UI thread unblocked.

`JavaScript` `Electron` `Node.js`

### [Delta — EECS-3311 Capstone](https://github.com/Abdullahu5mani/delta)
Software design course project at York University. Applied OOP design patterns to build a structured, maintainable Java system.

`Java` `OOP` `Design Patterns`
